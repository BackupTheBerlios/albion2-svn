Howto create an A2M Model?
with 3D Studio Max:
1. download an obj exporter for 3ds max (i.e. from http://www.habware.at/duck4.htm (named max2obj))
2. unzip the files and put them into your 3ds max plugin directory
3. start 3ds max and load a model
4. use file->export and select "Wavefront Object (*.OBJ)" as file type
5. a little window opens -> select "Group by: Object", "Use Materials", "Create Material Library", "Rotate Model", "Faces: Triangles",
"Texture Coordinates", "Smooth Groups". Deselect "Normals", "Compress Numbers" (?) and "Relative Vertex Numbers".
Put an "8" (or higher/lower) into "# of Digits" input field.
6. press OK
7. a second window opens -> select "Scene Materials" and "Export Maps". Deselect the rest and put a "1" into the "# of Digits" input field.
8. press OK
9. the obj file has been exported now. open the shell/command, move to the directory where the obj file or the obj2a2m program is.
10. type "obj2a2m model.obj model.a2m"
11. you should see some kind of that:
F:\Programme\dev\albion2\tools>obj2a2m ground.obj ground.a2m
1089 vertices, 2048 indices and 6144 texture vertices successfully converted!
12. et voila, now you have an a2m model that you can open with the engine
